Zadanie teoretyczne

  1. Wymień cztery sposoby wyprowadzania danych w JS

    • z użyciem document.write() – wypisuje dane bezpośrednio w dokumencie
    • z użyciem alert() – wyświetla komunikat w okienku dialogowym
    • do pól formularza – umożliwia wstawianie danych do elementów input
    • do tabeli – umożliwia prezentację danych w układzie tabelarycznym

  2. Czym jest funkcja w JS?

    • To zbiór instrukcji, które można wykonać poprzez wywołanie nazwy funkcji
    • Po wywołaniu funkcja wykonuje swój wewnętrzny kod krok po kroku
    • Może przyjmować argumenty, które są wykorzystywane do obliczeń, np. suma(a, b)
    • Może (ale nie musi) zwrócić wartość za pomocą słowa kluczowego return

  3. Omów metody getElementsByName, getElementById, getElementsByTagName

    Metoda Opis
    getElementById("id") zwra-ca pierwszy element o podanym atrybucie id
    getElementsByTagName("znacznik") zwraca kolekcję elementów o podanym typie znacznika, np. P, DIV
    getElementsByName("nazwa") zwraca elementy posiadające określoną wartość atrybutu name

  4. Omów właściwości innerHTML, innerText, className

    a) innerHTML - umożliwia odczyt lub zmianę kodu HTML znajdującego się wewnątrz elementu
    b) innerText - pozwala ustalić tekst umieszczony w elemencie, ignorując kod HTML
    c) className - określa nazwę klasy CSS odpowiedzialnej za wygląd elementu
